# Technology Stack

## Development Environment

- IDE: Visual Studio Code with Kiro AI assistant
- Platform: Windows (win32)
- Shell: bash

## Web Technologies

- HTML5 for semantic markup
- CSS3 for styling and responsive design
- Vanilla JavaScript for interactivity
- No build tools required - pure static files

## Development Workflow

### Local Development
```bash
# Serve locally (Python 3)
python -m http.server 8000

# Or use VS Code Live Server extension
# Right-click index.html > Open with Live Server
```

### Version Control
```bash
git status          # Check git status
git add .           # Stage changes
git commit -m ""    # Commit changes
git push            # Push to GitHub
```

## Deployment

- GitHub Pages (free static hosting)

## Best Practices

- Mobile-first responsive design
- Semantic HTML elements
- CSS custom properties for theming
- Iterative Development
- Accessibility (ARIA labels, keyboard navigation, also consider ADA compliance)
- Low complexity
- Only include lines of code that are essential to the function of the website
- Kepp files coherent for low-skill web app developers
