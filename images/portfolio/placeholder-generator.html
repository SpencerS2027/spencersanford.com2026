Portfolio Image Placeholder Generator

Instructions

This page generates placeholder images for the portfolio website. Click the download buttons to save each image in both JPEG and WebP formats.

  1. Click "Download JPEG" for each image below
  2. Use an online converter like Squoosh to convert JPEG to WebP
  3. Or use the command: cwebp -q 80 input.jpg -o output.webp
  4. Place both files in the images/portfolio/ directory

Note: These are simple placeholders. Replace with actual artwork before deployment.
